# Contextual Pre-Planning

## Installation

clone the repo:
```bash
git clone https://github.com/CLAIR-LAB-TECHNION/C-PREP
cd C-PREP
```

Create and activate anaconda environment:
```bash
conda env create -f environment.yml
conda activate rmrl
```

## Run:
see all options:
```bash
python -m rmrl --help
```
